# ControlForge Structured Text v1.2.5

**Release Date:** February 18, 2026

## Overview

Version 1.2.5 rewrites the AST parser to support multi-line variable declarations, fixes several critical bugs from the initial LSP launch, and cleans up legacy code and documentation.

## What's Fixed

- **Multi-line declarations**: Parser now handles declarations spanning multiple lines (#41)
- **Duplicate completions**: Removed redundant completion provider from extension client (#37)
- **Extension packaging**: iec61131-definitions/ now accessible in packaged .vsix (#39)
- **Workspace portability**: Removed hardcoded directory name lookup (#40)
- **Case sensitivity**: Symbol lookup normalized per IEC 61131-3 (#42)
- **Memory leak**: Index cleanup on file close, debounced change handling (#43)

## What's New

- AST parser rewrite with multi-line statement accumulator
- 29 AST parser unit tests (44 total)
- Clean build steps (`rm -rf out/dist` before compile/webpack)

## What's Removed

- Legacy parser and its test suite
- Stale documentation and scripts
- Stale git branches

## Installation

Install directly from the Visual Studio Code Marketplace or download the VSIX file from our GitHub releases page.
